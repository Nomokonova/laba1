# laba1
лабораторная работа 

Проект представляет собой набор из трех программ на C++, реализующих вычисление геометрических параметров прямоугольника, треугольника и трапеции. Каждая программа запрашивает у пользователя исходные данные, проверяет их корректность и выводит результаты расчетов.

Программа 1: Прямоугольник

Вычисляет периметр, площадь и диагональ по длине и ширине. Реализованы отдельные функции для каждой операции, выполняется проверка положительности сторон.

Программа 2: Треугольник

По трем сторонам проверяет существование треугольника, определяет равнобедренность, вычисляет периметр и площадь по формуле Герона.

Программа 3: Трапеция

По двум основаниям и боковым сторонам проверяет существование фигуры, вычисляет высоту через теорему Пифагора, затем находит периметр, площадь и среднюю линию.
